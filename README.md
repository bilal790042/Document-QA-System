This is a Langchain Project where concepts of chains, document loading, chunking, retrievers, llm calling and using Huggingface is used with frontend integrated with it.
Here FastAPI is used for api creation and react is used for frontend.

The projcet works withs uploading document of type 'txt, pdf and .docx' and then asking about it. The QA system will answer it.
Here MinMaxAI/MinMax-M2 is used a free model. Thanks to Huggingface for providing it to us.
